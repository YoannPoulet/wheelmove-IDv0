Introduction à l’analyse des chronogrammes

Bienvenue sur cette interface d’analyse des acquisitions locomotrices. Cette application vous permettra de :

  • Charger vos fichiers CSV contenant les données de vitesses linéaire et angulaire.
  • Analyser et segmenter automatiquement les tâches locomotrices.
  • Visualiser les chronogrammes et les différentes phases d’activité.
  • Sauvegarder les résultats pour un traitement ultérieur.

Avant de commencer, il est important de :

  • Lire attentivement les instructions de chargement des fichiers.
  • Vérifier les paramètres de seuil pour une segmentation correcte.
  • Comprendre le fonctionnement des différentes visualisations.

Une fois que vous aurez pris connaissance de toutes ces informations, cliquez sur les boutons ci-dessous pour accéder à l’application principale.
